# 聖經線上查詢應用 - 使用指南 (Usage Guide)

## 📖 使用者介紹 (User Introduction)

本應用提供一個簡單、快速、高效的聖經經文查詢平台。無論您是想要進行聖經研究、靈修或日常閱讀，本應用都能幫助您輕鬆找到所需的經文。

## 🎯 主要功能 (Main Features)

### 1. 聖經閱讀 (Bible Reading)

進入「聖經閱讀」頁面，您可以：
- 選擇聖經書卷（舊約/新約）
- 選擇特定章節
- 逐節閱讀聖經內容
- 舒適的排版和字體設置

**操作步驟：**
1. 點擊首頁導航欄中的「聖經閱讀」
2. 從列表中選擇要閱讀的書卷
3. 選擇具體的章節
4. 開始閱讀

### 2. 搜尋經文 (Search Verses)

「搜尋經文」功能讓您快速找到符合條件的經文。

**搜尋方式：**
- **關鍵字搜尋** - 輸入經文中的詞語或短語
- **主題搜尋** - 按主題（如：愛、信心、希望等）搜尋
- **人物搜尋** - 搜尋特定人物相關的經文
- **經文參考搜尋** - 直接輸入經文參考（如：約翰福音 3:16）

**操作步驟：**
1. 點擊導航欄中的「搜尋經文」
2. 在搜尋框中輸入搜尋關鍵字
3. 按下 Enter 或點擊搜尋按鈕
4. 瀏覽搜尋結果
5. 點擊結果查看完整經文

## 💡 使用技巧 (Tips & Tricks)

### 搜尋技巧

1. **精確搜尋** - 使用引號搜尋精確短語
   - 例如：`"神愛世人"`

2. **多關鍵字搜尋** - 使用空格分隔多個關鍵字
   - 例如：`愛 信心 盼望`

3. **排除詞語** - 使用減號排除特定詞語
   - 例如：`愛 -人`（搜尋含有「愛」但不含「人」的經文）

4. **主題搜尋** - 按主題分類查詢
   - 救贖、恩典、盼望、禱告、信心等

### 閱讀技巧

1. **章節跳轉** - 快速跳轉到特定章節
2. **書籤功能** - 標記重要經文供日後查閱
3. **複製功能** - 輕鬆複製經文內容
4. **分享功能** - 分享喜愛的經文

## ❓ 常見問題 (FAQ)

### Q: 搜尋沒有結果怎麼辦？
**A:** 請嘗試：
1. 檢查是否有打字錯誤
2. 使用更簡單或更具體的關鍵字
3. 嘗試用不同的詞語搭配

### Q: 如何收藏我喜歡的經文？
**A:** 點擊經文旁的書籤圖標，經文將被保存到您的收藏列表。

### Q: 應用可離線使用嗎？
**A:** 此應用需要網路連接。建議將其加入書籤以便快速存取。

### Q: 如何在手機上使用？
**A:** 本應用完全響應式設計，在任何設備上都能正常使用。建議在手機首屏添加快捷方式。

### Q: 如何報告問題或提出建議？
**A:** 請訪問 GitHub 倉庫提出 Issue 或 Pull Request。
