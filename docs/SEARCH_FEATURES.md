# 聖經查詢功能指南 (Bible Search Features Guide)

本文檔說明如何充分利用本應用的搜尋功能。

## 🔍 搜尋方式 (Search Methods)

### 1. 關鍵字搜尋 (Keyword Search)

最基本的搜尋方式 - 輸入您要查找的詞語或短語。

**例子：**
- 搜尋「愛」- 找到所有包含「愛」的經文
- 搜尋「信心」- 找到所有關於信心的經文

### 2. 經文參考搜尋 (Verse Reference Search)

使用書卷名稱、章節和節數進行搜尋。

**格式：** 書卷名 章:節

**例子：**
- 約翰福音 3:16
- 詩篇 23:1
- 羅馬書 8:28

### 3. 短語搜尋 (Phrase Search)

搜尋特定的完整短語。

**例子：**
- "神愛世人" - 找到這個確切的短語
- "聖經" - 找到所有包含此詞語的經文

### 4. 多詞搜尋 (Multi-word Search)

同時搜尋多個詞語。

**例子：**
- 神 愛 世人 - 找到包含這些詞語的經文

## 📚 主題搜尋 (Thematic Search)

按照以下主題分類搜尋經文：

### 信仰主題 (Faith Themes)
- **信心** - 關於相信和信仰的經文
- **盼望** - 關於未來希望的經文
- **愛** - 關於愛的各種形式
- **恩典** - 關於上帝恩典的經文
- **救贖** - 關於救贖的經文

### 人生主題 (Life Themes)
- **禱告** - 禱告的重要性和方法
- **智慧** - 智慧和知識的經文
- **饒恕** - 寬恕和和解
- **試煉** - 面對困難和挑戰
- **喜樂** - 快樂和滿足

### 道德主題 (Moral Themes)
- **順服** - 服從上帝的命令
- **誠實** - 真實和誠實
- **忍耐** - 堅持和耐心
- **品格** - 道德和品格發展

## 💡 搜尋技巧 (Search Tips)

### 精確搜尋
使用引號搜尋精確短語：
```
"神愛世人"
```

### 排除詞語
在詞語前加減號來排除結果：
```
聖經 -舊約  # 搜尋含有「聖經」但不含「舊約」的經文
```

### 模糊搜尋
搜尋相似但不完全相同的詞語：
```
信心 信 信德  # 搜尋任何相似的詞語
```

## 🌟 高級搜尋 (Advanced Search)

### 與操作符 (AND)
搜尋必須包含所有詞語：
```
信心 AND 愛 AND 盼望
```

### 或操作符 (OR)
搜尋包含任何詞語：
```
喜樂 OR 快樂 OR 歡喜
```

### 組合搜尋 (Combined)
```
(信心 OR 相信) AND (神 OR 上帝)
```

## 🔔 搜尋提示 (Search Hints)

- **慢慢開始** - 先用簡單的單詞，然後根據需要細化
- **檢查拼寫** - 確保正確的拼寫和繁簡體轉換
- **嘗試變體** - 如果第一個搜尋沒有結果，嘗試相似的詞語
- **查看上下文** - 閱讀整個段落以獲得更好的理解
- **使用書籤** - 標記常見的搜尋結果供日後參考
