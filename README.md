# 聖經線上查詢應用 (Bible Search Application)

![聖經閱讀](https://img.shields.io/badge/language-TypeScript-blue) ![Vue3](https://img.shields.io/badge/framework-Vue%203-green) ![Vite](https://img.shields.io/badge/build-Vite-purple)

## 📖 專案介紹 (Project Overview)

**聖經線上查詢應用**是一個現代化的 Web 應用程式，提供快速便捷的聖經經文查詢和閱讀功能。使用 Vue 3、TypeScript 和 Vite 構建，具有優秀的性能和使用者體驗。

**Bible Search Application** is a modern web application that provides quick and convenient Bible verse search and reading capabilities. Built with Vue 3, TypeScript, and Vite for excellent performance and user experience.

### ✨ 主要功能 (Key Features)

- 📚 **聖經全文閱讀** - Browse complete Bible chapters
- 🔍 **快速搜尋** - Fast verse search functionality
- 📱 **響應式設計** - Mobile-friendly interface
- ⚡ **高效能** - Lightning-fast search and navigation
- 🌍 **多語言支援** - Traditional Chinese interface

## 🚀 快速開始 (Quick Start)

### 環境需求 (Requirements)
- Node.js 14+
- npm 或 yarn

### 安裝步驟 (Installation)

```bash
# 克隆此倉庫
git clone https://github.com/hmlu36/vite-bible.git

# 進入專案目錄
cd vite-bible

# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev
```

專案將在 `http://localhost:5173` 運行

## 🛠️ 開發 (Development)

### 可用命令 (Available Commands)

```bash
# 開發伺服器 (Dev server)
npm run dev

# 構建生產版本 (Build for production)
npm run build

# 預覽生產構建 (Preview production build)
npm run preview
```

## 📖 使用指南 (Usage Guide)

詳細的使用指南請參考 [USAGE_GUIDE.md](./USAGE_GUIDE.md)

### 基本使用 (Basic Usage)

1. **聖經閱讀** - 點擊導航欄的「聖經閱讀」進入主頁面瀏覽聖經內容
2. **搜尋經文** - 點擊「搜尋經文」使用關鍵字搜尋聖經內容
